# CyberSecurity-Msc-Portfolio
This repository contains cybersecurity projects completed during my MSc Cyber Security at Cardiff University.

The portfolio demonstrates practical experience in:

- Penetration Testing
- Malware Analysis
- Digital Forensics
- Cybersecurity Operations
- Secure Software Development
- Risk Management

Each project includes methodology, tools used, findings, and remediation strategies.
